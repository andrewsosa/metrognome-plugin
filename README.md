# Metrognome

A metronome that still counts every beat. 

Increasing the tick count just inserts more tocks between your ticks:
  - `1`: All ticks (tick, tick, tick...)
  - `2`: Alternate tick/tock (tick, tock, tick, tock...)
  - `3`: Tick + 2 tocks (tick, tock, tock, tick, tock, tock...)
  - And so on...
